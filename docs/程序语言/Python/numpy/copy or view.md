Q1 从list构造array，使用复制还是视图？

A1 如果list中是相同基本数据类型（int, float等），则采用copy，否则采用view。因为如果list保存如字典这类数据，np的数组没法以连续内存保存这些数据，因此不得不采用view形式。

```python
>>> import numpy as np
>>> a = [[1, 2], [3, 4]]
>>> b = np.array(a)
>>> b
array([[1, 2],
       [3, 4]])
>>> b[0][0] = 0
>>> a
[[1, 2], [3, 4]]
>>> b
array([[0, 2],
       [3, 4]])
>>> a = [{'a': 1, 'b':2}, {'a': 3, 'b': 4}]
>>> b = np.array(a)
>>> b[0]['a'] = 0
>>> a
[{'a': 0, 'b': 2}, {'a': 3, 'b': 4}]
```
