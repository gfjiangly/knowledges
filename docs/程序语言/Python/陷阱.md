## math.pow返回浮点数

```python
>>> int(math.pow(10, 28))
9999999999999999583119736832
```

转成整形发现不对了，这是因为浮点数表示精度问题，使用 `10 ** 28`计算指数



## 创建二维list时浅拷贝

```python
>>> a = [[0] * 3]* 2
>>> a[0][0] = 1
>>> a
[[1, 0, 0], [1, 0, 0]]
```

需要使用 `for in` 创建，此时是深拷贝

```python
>>> a = [[0 for _ in range(3)] for _ in range(2)]
>>> a[0][0] = 1
>>> a
[[1, 0, 0], [0, 0, 0]]
```
