## python float

#  round(2.634, 2)
\# 2.63 常规，四舍五入
round(2.635, 2)
\# 2.63 碰到5，奇数后面的直接舍弃
round(2.665, 2)
\# 2.67碰到5，偶数向上取舍

#  格式化（得到的是字符串）
‘{:.2f}’.format(2.6342344556)
或者‘%.2f’ % 2.6342344556


## numpy

np.set_printoptions(precision=3)
注意这只是数据显示的时候的小数位置，写入到文件发现还是原来的位数

要想真正四舍五入的话使用函数：
numpy.around(a, decimals=0, out=None)
Evenly round to the given number of decimals.
